#### ECharts

[Handbook - Apache ECharts](https://echarts.apache.org/handbook/zh/get-started/)

##### 1、下载

```
https://www.jsdelivr.com/package/npm/echarts

选择dist/echarts.js并保存为 echarts.js 文件（保存在项目中当源码使用）
```

##### 2、引入

```
html中
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8" />
    <!-- 引入刚刚下载的 ECharts 文件 -->
    <script src="echarts.js"></script>
  </head>
</html>
```
